# Structure Buildify - Next.js Clone

This is a modern, high-performance clone of [structurebuildify.com](https://structurebuildify.com/) built with Next.js 14, TypeScript, and CSS Modules.

## Features

-   **Next.js 14 App Router:** Leveraging the latest features of Next.js.
-   **TypeScript:** Type-safe development.
-   **Vanilla CSS (CSS Modules):** Clean, scoped styling following the original site's aesthetic.
-   **Lucide React:** Modern and consistent iconography.
-   **Responsive Design:** Fully optimized for mobile, tablet, and desktop.
-   **SEO Optimized:** Meta tags and semantic HTML.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

-   `app/`: Contains the pages and global styles.
-   `components/`: Reusable UI components.
-   `public/`: Static assets.
